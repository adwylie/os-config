# Configuration

## Applications

- vscodium
  - color theme
  - extensions
- zsh brew completions
- iTerm2
  - Profiles > Colors > Color Presets...
  - Profiles > Keys > Key Bindings > Presets... > Natural Text Editing
- musicbrainz-picard
  - destination directory
  - file naming script:
    ```
    $if2(%albumartist%,%artist%) - %album% $if(%originalyear%,\(%originalyear%\),)/
    $if($gt(%totaldiscs%,1),%discnumber%-)$num(%tracknumber%,2). %artist% - %title%
    ```


## Media

When setting up "Movies", "Music", and "Pictures" open the corresponding MacOS application
and ensure that files will not be moved or copied to the libraries.
Instead, create a "content" directory under each of the media directories to contain our media.
This folder can then be directly backed up to/from Borg,
and organized by 3rd-party tagging applications (such as MusicBrainz Picard).
